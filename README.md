# SealDice COC7th 自动化辅助插件

本仓库包含一组基于 `sealdice-core` 的 JavaScript 插件，旨在为 COC7th（克苏鲁的呼唤第七版）跑团提供更自动化的规则检定支持，减少守密人（KP）的手动记账负担。

### 测试环境
* **核心版本**：`sealdice-core 1.5.1`
* **运行平台**：`darwin-arm64` (macOS Apple Silicon)
* *注：理论上兼容其他平台的同版本核心，如有兼容性问题欢迎反馈。*

---

## 包含功能

### 1. 临时疯狂自动检定 (`.sc` 辅助)
* **核心逻辑**：基于 COC7th 规则。
* **自动化流程**：插件会持续监听理智检定（San Check）。当单次扣除的理智值**超过 5 点**时，插件会自动唤起一个独立的**智力检定**，并根据检定结果自动给出对应的状态提示（是否陷入临时疯狂）。

### 2. 重伤与死亡豁免检定 (`.st` 辅助)
* **核心逻辑**：基于 COC7th 伤残与濒死规则，通过监听 `.st` 属性变更指令实现。
* **重伤判定**：当检测到角色的当前 `hp` 单次扣减**超过 `maxhp` 的一半**时，自动触发**体质检定**，并在检定后自动将该角色标记为【重伤】状态。
* **状态联动**：
  * **重伤移除**：角色在受到治疗后，可手动将【重伤】标记移除。
  * **死亡豁免**：当角色 `hp` 归零时，插件会自动检测其是否带有【重伤】标记，以此来智能判定角色是直接进入【濒死】状态还是触发其他濒死流程。

---

## 安装与使用
1. 下载本仓库的 `.js` 插件文件。
2. 将文件放入海豹核心（SealDice）的 `plugin` 对应 JavaScript 插件目录下。
3. 重启海豹核心或在后台重新加载插件即可生效。
